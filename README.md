# Fiche-maladie-PA
Fiche maladie: la Pancréatite aigüe.

Application ayant pour but de générer des rapports (pdf) pour les patients souffrant de Pancréatite aigüe.

 * NB: Stockage des rapports sous: "Documents\Fiches Pancréatite".
